
Emacs
=====

Install Emacs via:

    $ sudo apt-get install emacs


Commands
--------
 
C => Strg  
M => Alt  

C-c C-b		Beautify Buffer  
C-w 		Cut Text  
M-w			Copy Text  
C-y			Paste Text  
C-space		Set mark  
C-x C-s		Save File  
C-X u		Undo

C-c C-p C-w		Copy entity or component
C-c C-p C-c		Paste as component
C-c C-p C-i		Paste as instance
C-c C-p C-s		Paste as signals
